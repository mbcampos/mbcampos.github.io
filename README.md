# mbcampos.github.io

Projetos hospedados via GitHub Pages.

## folhaponto
Gerador de folha de ponto mensal imprimível (formato A4) com dados de entrada, saída e almoço para todos os dias úteis. Permite exportar para PDF com `html2pdf.js`.


## juroscompostos
Simulador financeiro com dois modos: calcular valor final (projeta saldo a partir de aportes) e planejar investimento (calcula taxa necessária ou valor inicial para atingir uma meta), com gráfico de evolução e opção de desconto de IR.
